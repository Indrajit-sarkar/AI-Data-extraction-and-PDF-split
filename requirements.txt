# ============================================================================
# PLATFORM-SPECIFIC NOTES
# ============================================================================
#
# WINDOWS:
# - All dependencies work out-of-the-box
# - pdf2image automatically handles poppler-utils
# - No additional system dependencies required
# - Full barcode detection support included
#
# MACOS:
# - Requires system dependencies: brew install poppler zbar
# - Virtual environment recommended: python3 -m venv venv
# - M1/M2 Macs: All packages have ARM64 support
# - Use automated installer: ./install-macos.sh
#
# LINUX:
# - Requires system dependencies:
#   Ubuntu/Debian: sudo apt-get install poppler-utils libzbar0 libgl1-mesa-glx
#   CentOS/RHEL: sudo yum install poppler-utils zbar mesa-libGL
# - Virtual environment recommended: python3 -m venv venv
# - Use automated installer: ./install-linux.sh
#
# ============================================================================
# INSTALLATION COMMANDS BY PLATFORM
# ============================================================================
#
# WINDOWS (Command Prompt):
# python -m pip install --upgrade pip
# python -m pip install -r requirements.txt
#
# MACOS/LINUX (Terminal):
# python3 -m pip install --upgrade pip
# pip install -r requirements.txt
#
# WITH VIRTUAL ENVIRONMENT (Recommended for macOS/Linux):
# python3 -m venv venv
# source venv/bin/activate  # macOS/Linux
# # or venv\Scripts\activate  # Windows
# pip install -r requirements.txt
#
# ============================================================================
# TROUBLESHOOTING
# ============================================================================
#
# IF INSTALLATION FAILS:
# 1. Upgrade pip: python -m pip install --upgrade pip setuptools wheel
# 2. Install individually: pip install package_name
# 3. Use --user flag: pip install --user -r requirements.txt
# 4. Check system dependencies (macOS/Linux only)
#
# IF BARCODE DETECTION FAILS:
# - Windows: Should work automatically
# - macOS: brew install zbar
# - Linux: sudo apt-get install libzbar0 (Ubuntu) or equivalent
#
# IF OPENCV FAILS:
# - Try headless version: pip install opencv-python-headless
# - Install system dependencies first (Linux/macOS)
#
# IF AZURE LIBRARIES FAIL:
# - Check internet connection
# - Try: pip install --upgrade azure-ai-formrecognizer azure-storage-blob
#
